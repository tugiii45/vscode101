# Elite Business Consulting Landing Page #
## Project Brief ##
 
This project is a responsive landing page for a fictional consulting firm called _Elite Business Consulting Page_. The goal of the project is to create a professional web presence for a business using semantic HTML and modern CSS layout techniques.

The website includes a hero section that introduces the company, a services section describing what the business offers, an about section explaining the company's mission, and a contact form where potential clients can reach out.

The project simulates a real client request where a company needs a clean, professional, and responsive website that works across different devices and browsers.
## Technologies Used ##
- HTML5- For structuring the website 
- CSS- For styling the page
- CSS Grid- For creating responsive layouts
- GitHub- For hosting repository
- GitHub Pages- For deployment of the live website

## Setup Instructions ##
To run this project on your computer:

1. Clone the repository
```https://github.com/tugiii45/vscode101.git```
2. Open the project folder in Visual Sudio Code.
3. Open index.html file.
4. Use **Live Server** to view the website

## Business Rationale ##

The website is designed to represent a professional consulting firm that helps businesses grow through strategy, research, and digital transformation.

The landing page focuses on clear communication of the company's value proposition, making it easy for potential clients to understand the services offered and contact the company.

The layout is mobile-first and responsive to ensure accessibility across devices. Semantic HTML and SEO-friendly structure were used to improve usability, accessibility, and search engine visibility.
